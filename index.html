Two Different Visualizations:

  • Dot Density Map: Each dot represents a collision, color-coded by severity.
  • Heatmap: Quickly spot high-risk areas with intense traffic incidents.

Features & Functionality

  • Interactive Web Map: Built using Mapbox for smooth visualization. Displays collision locations with pop-up details.
  • User-Controlled Filters: Filter crash data by year, type (vehicle, pedestrian, cyclist), severity, and contributing factors.
  • Heatmaps & Clustering: Highlights high-risk areas to identify traffic hazards.
